Design Concept

Connection Philosophy

The logo represents the core mission: connecting people through sign language.

Two circles symbolize two parties (hands, people, nodes) establishing contact.

The bridge line is the intentional connection between them.

Signal arcs show data and communication flowing through this bridge.

Minimal, geometric, and meaningful — every element serves a purpose.

Design Specifications

Font Family
System default (SF Pro Display on Apple, Segoe UI on Windows)
Font Weight
600 (Semi-bold) for modern, premium feel
Letter Spacing
-0.5px for tighter, contemporary look
Icon Nodes
Circles with radius 5.5, opacity 90-95% for depth
Bridge Line
Stroke width 2.5, full opacity, rounded caps
Signal Arc
Subtle accent (40-65% opacity) showing data flow
Color Scheme
Uses currentColor for theme support (light/dark)
Hover State
80% opacity with smooth transition (150ms)

Brand Values
